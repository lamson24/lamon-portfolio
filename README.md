# LAMON-DIY Landscape Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Publish

This site can be published as a static site. Use the project root as the publish directory and leave the build command empty.

Required files and folders:

- `index.html`
- `project.html`
- `style.css`
- `script.js`
- `project.js`
- `canvas.js`
- `content.js`
- `assets/`
- `locales/`

## Notes

The contact form uses FormSubmit and sends inquiries to `Lamsonarchp@gmail.com`. FormSubmit sends a confirmation email the first time the form is submitted; click that confirmation link once to activate delivery.
